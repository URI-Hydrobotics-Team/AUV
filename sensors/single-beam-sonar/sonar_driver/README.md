# Single Beam Sonar

Driver written for the Single Beam Sonar

# Dependencies
- [bluerobotics ping-cpp library](https://github.com/bluerobotics/ping-cpp) -> use the Master branch
- Boost (libboost-all-dev) -> used for generating files for the library


# Usage
- Need to modify the header file in driver.h to the correct path
- Might need to modify the header file and library location within the ping-cpp library
